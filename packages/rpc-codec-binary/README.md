# `@jsonjoy.com/rpc-codec-binary`

Binary codec for [JSON Reactive RPC](https://jsonjoy.com/specs/json-rx)
messages. Encodes and decodes RPC messages using the
[binary encoding](https://jsonjoy.com/specs/json-rx/binary-encoding).

## Usage

```ts
import {BinaryMsgStreamCodec} from '@jsonjoy.com/rpc-codec-binary';
```
