# `@jsonjoy.com/rpc-codec-json-rpc-2`

JSON-RPC 2.0 codec for [JSON Reactive RPC](https://jsonjoy.com/specs/json-rx)
messages. Encodes and decodes RPC messages using the standard
[JSON-RPC 2.0](https://www.jsonrpc.org/specification) format.

## Usage

```ts
import {JsonRpc2TypedMsgStreamCodec} from '@jsonjoy.com/rpc-codec-json-rpc-2';
```
