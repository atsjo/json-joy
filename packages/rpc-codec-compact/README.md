# `@jsonjoy.com/rpc-codec-compact`

Compact codec for [JSON Reactive RPC](https://jsonjoy.com/specs/json-rx)
messages. Encodes and decodes RPC messages using the
[compact JSON encoding](https://jsonjoy.com/specs/json-rx/json-encoding).

## Usage

```ts
import {CompactMsgStreamCodec} from '@jsonjoy.com/rpc-codec-compact';
```
