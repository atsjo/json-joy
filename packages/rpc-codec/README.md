# `@jsonjoy.com/rpc-codec`

Aggregator package for all [JSON Reactive RPC](https://jsonjoy.com/specs/json-rx)
message codecs. Re-exports `RpcCodec`, `RpcCodecs`, `RpcMessageCodecs`, and
`JsonCompactMsgCodec`.

## Usage

```ts
import {RpcCodec, RpcMessageCodecs, RpcCodecs} from '@jsonjoy.com/rpc-codec';
```
